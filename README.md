# GeometrySharp

### 🔺🔹This library allows you to use Geometry formulas◽⚪

## Installation
- **type this command:**
```
dotnet add package GeometrySharp
```
## How to use
- **type this to use GeometrySharp:**
```csharp
using GeometrySharp;
```
## Example
```csharp
double RectArea = PlaneGeometry.Rect.GetArea(2, 4); // Output: 8
double CubeSurface = SolidGeometry.Cube.GetSurfaceArea(6); // Output: 216
```
