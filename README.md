# GeometrySharp

![NuGet Version](https://img.shields.io/nuget/v/GeometrySharp?style=for-the-badge) ![NuGet Downloads](https://img.shields.io/nuget/dt/GeometrySharp?style=for-the-badge)
### 🔺🔹This library allows you to use Geometry formulas◽⚪

## Installation
- **type this command:**
```
dotnet add package GeometrySharp
```
## How to use
- **type this to use GeometrySharp:**
```csharp
using GeometrySharp;
```
## Example
```csharp
double RectArea = PlaneGeometry.Rect.GetArea(2, 4); // Output: 8
double CubeSurface = SolidGeometry.Cube.GetSurfaceArea(6); // Output: 216
```
